Architecture

What we built on top of the original FunASR engine.

🎤 Microphone                                               
  │  ffmpeg (1.5s chunks)
  ▼
🔇 VAD Filter (fsmn-vad)
  │
  ▼
🧠 SenseVoiceSmall (ASR Engine)
  │
  ├──→ 📝 Punctuation Model (punc_ct-transformer)
  ├──→ 🌍 Google Translate (20+ languages)
  └──→ 🤖 Ollama (AI mindmap/summary/quotes)
  │
  ▼
⚡ SSE Stream → Browser (Real-time <100ms)
  │
  ├──→ 📥 SRT Export (subtitles with timestamps)
  ├──→ 🎙️ WAV Recording (save to desktop)
  └──→ 🎨 20 Themes (CSS variable system)

Quick Start

Three commands. That's it.

# 1. Install dependencies
pip install funasr deep-translator numpy

# 2. (Optional) Install Ollama for AI summarization
brew install ollama && ollama pull qwen2.5:14b

# 3. Run
python3 funasr_live.py

# 4. Open
open http://localhost:8765
